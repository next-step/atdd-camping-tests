# 실행 방법

## 테스트 실행 방법
### 레포지토리 clone
```bash
./gradlew repo-clone
```

### 애플리케이션 실행
```bash
./gradlew app-up
```

### 테스트 실행
```bash
./gradlew test
```

<br>

## 종료 방법

### 애플리케이션 종료
```bash
./gradlew app-down
```

### 데이터베이스 종료
```bash
./gradlew db-down
```

<br>

## 상태 확인 방법

### 애플리케이션/데이터베이스 상태 확인
```bash
./gradlew ps
```

<br>

## 로그 확인 방법
### kiosk 로그 확인
```bash
./gradlew kiosk-logs
```

### admin 로그 확인
```bash
./gradlew admin-logs
```

### reservation 로그 확인
```bash
./gradlew reservation-logs
```
